// Authentication Required
JWT > Dashboard: Authenticated access
JWT > Categories: Manage categories
JWT > Transactions: Financial operations

// Category Setup Flow
Categories > DefaultSetup: First time setup
DefaultSetup > CustomCategories: Add personal categories
CustomCategories > Transactions: Ready to track finances

// Transaction Creation Flow
TransactionFlow [label: "Transaction Creation"] {
  SelectType [icon: choice, label: "Select Type (Income/Expense)"]
  SelectCategory [label: "Choose Category"]
  EnterAmount [icon: calculator, label: "Enter Amount"]
  SelectPayment [icon: credit-card, label: "Payment Method"]
  AdvancedOptions [label: "Advanced Options"]
}

Transactions > TransactionFlow: Create new transaction
SelectType > SelectCategory: Define transaction
SelectCategory > EnterAmount: Categorize expense/income
EnterAmount > SelectPayment: Set amount
SelectPayment > AdvancedOptions: Choose payment method

AdvancedOptions > InterestOptions: Interest/tax calculation
